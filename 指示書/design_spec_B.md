# かんたんツールボックス — デザイン指示書（B案）

---

## 1. デザインコンセプト

**フロスト・ジオメトリ（Frost Geometry）**

白と青を基調とした清潔感あるデザイン。深みのあるネイビーのヘッダーと、ライトブルーの背景面が対比を生む。カード型レイアウトで情報をグループ化し、視認性と操作性を両立する。

## 4. レイアウト・構造

### パネル全体

- **幅：** 約280–320px（固定幅）
- **背景：** `#f5f9ff`
- **ボーダー：** `1px solid #c5dcf5`
- **角丸：** `20px`
- **overflow：** `hidden`

### ヘッダー

- **背景：** `#0c51a0`
- **パディング：** `16px 16px 0`
- **右上アクセント：** `#1a6fd4`、幅70px × 高さ70px、左下のみ角丸`70px`、`opacity: 0.45`。ヘッダー右上角に `position: absolute` で配置

#### ブランド名
- テキスト：`MEIYUSHA`（大文字）
- 色：`#7fb5e8`
- フォントサイズ：8px
- レタースペーシング：`0.25em`
- マージン下：3px

#### タイトル
- 「かんたん」：`font-weight: 300`、`color: #ffffff`
- 「ツールボックス」：`font-weight: 500`、改行して `display: block`
- フォントサイズ：16–17px
- マージン下：12px

#### タブバー
- ヘッダー末尾、背景透明
- タブ間に `gap: 2px`
- 各タブ：パディング `6px 8–10px`、フォントサイズ9–10px、色 `#7fb5e8`
- **選択タブ：** 背景 `#f5f9ff`（パネル背景と同色）、文字色 `#0c51a0`、`font-weight: 500`、角丸 `6px 6px 0 0`

### ボディ

- **背景：** `#f5f9ff`
- **パディング：** `12–14px`

### フッター

- **背景：** `#ffffff`
- **ボーダー上：** `1px solid #c5dcf5`
- **パディング：** `8–10px 12–14px`
- 3要素を横並び（`justify-content: space-between`）
  1. 「閉じる」ボタン（左）
  2. ズームコントロール（中央）
  3. ミニツールバートグル（右）

## 6. グリッドレイアウトのパターン

| パターン | 使用場面 |
|----------|----------|
| `1fr 1fr`（2列） | 対になるボタン・入力欄 |
| `1fr 1fr 1fr`（3列） | 3つ並ぶアクション（枠の種類など） |
| `repeat(4, 1fr)`（4列） | 記号グリッド |
| `1fr auto`（ラベル＋値） | 設定行（用紙サイズ等） |

## 7. インタラクション

- ボタンホバー：セカンダリボタンの背景を `#dce8f7` へ変化
- バッジホバー：背景 `#0c51a0`、文字 `#ffffff` へ変化
- タブ選択：選択タブのみ白背景＋ネイビーテキストへ切り替え（他タブは `#7fb5e8`）
- フォーカス時：ボーダー色を `#0c51a0` に変化させることを推奨

## 9. 注意事項

- ヘッダーの右上アクセント（ジオメトリ装飾）は `position: absolute` で配置し、テキストが前面に来るよう `position: relative` を親要素に付与する
- タブバーとボディの継ぎ目は、選択タブの背景色（`#f5f9ff`）がボディと同色になることで自然につながる
- パネル全体の `overflow: hidden` により、ヘッダーの角丸が保持される
- カードコンポーネントはボディの背景（`#f5f9ff`）より白く浮かせることで階層感を出す
